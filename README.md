# Gramzdas Iedzīvotāju Padomes Digitālais Centrs

![Status](https://img.shields.io/badge/Statuss-Aktīvs-success)
![Tehnoloģija](https://img.shields.io/badge/Metode-AI_Augmented-blueviolet)

Šis ir Gramzdas Iedzīvotāju Padomes oficiālais portāls — inovatīvs risinājums pašvaldības līmeņa komunikācijai. Portāls nodrošina caurspīdīgu, modernu un ērtu saziņu starp iedzīvotājiem un padomi.

## 🚀 Tehnoloģiskais Inovācijas Projekts

Projekts ir unikāls ar savu **"AI-First" izstrādes pieeju**. Izmantojot Mākslīgo Intelektu kā programmēšanas asistentu un sistēmu arhitektu, ir izveidots tehniski sarežģīts risinājums bez tradicionālas programmētāju komandas. Šī pieeja ļāva:
* Realizēt **dinamisku datu apmaiņu** starp mākoņpakalpojumiem.
* Izveidot modernu, responsīvu dizainu rekordīsā laikā.
* Nodrošināt sistēmas ilgtspēju ar minimālām uzturēšanas izmaksām.

## ✨ Galvenās Funkcijas

* **📰 Dinamiskas Aktualitātes:** Integrācija ar Google Sheets ļauj padomei publicēt ziņas bez tehniskām zināšanām.
* **🗺️ Problēmu Karte:** Interaktīva karte (Leaflet.js), kurā vizualizēti aktuālie ziņojumi par labiekārtošanas vajadzībām.
* **💡 Ideju Siena:** Reāllaika platforma ideju iesniegšanai un kopienas balsojumam, izmantojot Firebase datubāzi.
* **⏳ Darbu Laika Līnija:** Pārskatāms un vizuāls atskaitīšanās veids par padomes paveikto.
* **📊 Prezentāciju Režīms:** Iebūvēts rīks publiskām sanāksmēm ar animētu saturu un rīkiem.

## 🛠️ Sistēmas Arhitektūra

Projekts balstīts uz modernu "No-Code / Low-Code" datu pārvaldības struktūru:

* **Lietotāja saskarne (Frontend):** HTML5, CSS3 (Glassmorphism stils), Vanilla JavaScript.
* **Datu vadība (CMS):** Google Sheets API (datu ielādei un struktūrai).
* **Reāllaika dati:** Firebase Realtime Database (interaktīvajiem elementiem un balsojumiem).
* **Kartogrāfija:** Leaflet.js un OpenStreetMap integrācija.



## 📂 Projekta Struktūra

* `index.html` — Centrālais navigācijas mezgls.
* `aktualitates.html` — Ziņu siena (automātiska sinhronizācija ar tabulu).
* `karte.html` — Kartes modulis problēmu ziņošanai.
* `idejas.html` — Sabiedrības iesaistes rīks (balsošana).
* `darbi.html` — Vizuālā atskaite par darba gaitu.
* `/scripts/` — Sistēmas loģika un API integrāciju kodi.
